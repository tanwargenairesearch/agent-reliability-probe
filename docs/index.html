A measurement series

Enterprise Agent Reliability

How reliably do AI agents handle real business tasks — not just once, but every time? Each note is one finding, measured on real tasks, with open-source code and reproducible numbers.

Note 1

Capable isn't the same as dependable

Accuracy says an agent can do a task; reliability says it will, every time. The ~14-point gap, measured on 100 real customer-service tasks.

 Note 2

Most agent failures are the kind you can't reproduce

Across 1,500 runs and two domains, ~3 in 4 failures were intermittent — right on some runs, wrong on others. The hardest failures to catch are also the most common.
